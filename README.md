## 项目描述

这是一个简洁美观的个人导航起始页（StartPage），提供快速访问常用网站的功能。项目基于第三方代码进行二次开发，具有现代化的界面设计和流畅的用户体验。

**主要特性：**
- 响应式设计，适配各种屏幕尺寸
- 鼠标滑动特效（3.js文件提供）
- 自定义字体支持（上海东观体）
- 简洁的界面布局
- 快速加载和访问

**演示地址：** https://www.loser.dev

## 使用说明

### 快速部署

1. 将项目文件上传到您的Web服务器
2. 确保所有文件在同一目录下：
   - `index.html` - 主页面
   - `1.css` - 样式文件
   - `2.js` - 导航功能脚本
   - `3.js` - 鼠标特效脚本
   - 字体文件 - 可选，用于自定义字体显示

### 自定义配置

- 修改 `index.html` 中的导航链接来添加您常用的网站
- 调整 `1.css` 中的样式来自定义界面外观
- 如需更换字体，请替换字体文件并更新CSS中的字体引用

### 本地预览

直接在浏览器中打开 `index.html` 文件即可预览效果。

## 技术说明
